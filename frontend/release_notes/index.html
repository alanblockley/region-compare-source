Feature Release: Tool Kit & LLM Compare

Saturday, 1st June 2024

Now that Bedrock is expanding in region I find a common question asked is "What Large Language Models are in my region?". This new feature see's the addition of the Region Compare Tool Kit and it's first "Tool" is LLM Compare.

What's a "Tool"? - A tool is a more specific, targetted querty of cross region capability.

This release also see's some backend improvements in data structure and organisation
